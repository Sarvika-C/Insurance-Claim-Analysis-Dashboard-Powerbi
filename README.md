# 🛡️ Insurance Claim Analysis Dashboard (Power BI)

An interactive **Insurance Claim Analysis Dashboard** built in **Microsoft Power BI** to analyze insurance claim data and deliver actionable business insights. The dashboard leverages **Power Query**, **DAX**, and interactive visualizations to monitor revenue, profit, customer metrics, claim performance, and regional trends.

---

## 📌 Project Overview

The **Insurance Claim Analysis Dashboard** provides a comprehensive overview of insurance claim performance through dynamic reports and visualizations. It enables users to evaluate key business metrics, identify trends, compare regional performance, and make data-driven decisions using interactive filters and KPIs.

---

## 🚀 Features

- 📊 Interactive KPI Cards
- 💰 Revenue & Profit Analysis
- 👥 Total Customer Overview
- 📈 Year-wise Profit Trend Analysis

- 🌍 Revenue by Region
- 📦 Profit by Sub-Category
